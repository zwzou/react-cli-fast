# react-cli-fast

> 一个精简的，快速的react项目！

开始条件，电脑安装了node.js以及npm，最新版本的node.js里面自带npm。然后进入你要安装项目的目录，到cmd里面执行命令

// 全局安装脚手架<br>
`sudo npm install -g z-build`

// 下载react-cli-fast模板到你的目录，过程中可指定项目名称<br>
`sudo z-build init react-cli-fast`

// 初始化项目<br>
`sudo npm install`

// 开始项目<br>
`sudo npm start || npm run dev`
